# Swift Boilerplate

My custom template for Swift projects.

## Project structure

__* Base structure without pods__

```
└─ SwiftBoilerplate
   └─ SwiftBoilerplate
      └─ Util
      └─ Resources
      └─ Extension
      └─ Entity
      └─ Logic
      └─ Networking
      └─ Storage
      └─ Storyboard
      └─ Controller
```
